## Shortcuts

| --- | --- | --- |
| Search selection in new tab | `Ctrl+E` | - |
| Open new tab to the right | `Ctrl+T` | Chrome opens new tabs at the end of the strip, splitting up related tabs. This keeps new tabs next to the current one. |
